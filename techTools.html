Open source web servers

Over 80% of web applications and websites are powered by open source web servers. a web server is "a computer system that processes requests via HTTP, the basic network protocol used to distribute information on the World Wide Web. The term can refer to the entire system, or specifically to the software that accepts and supervises the HTTP requests." a. Apache HTTP server: The Apache HTTP Server often referred to as httpd, or simply Apache. It was first launched in 1995. Apache powers 52% of all websites globally, and is by far the most popular web server. b. NGINX: it is shorthand for “how do your design a web server which can handle ten thousand concurrent connections?” NGINX relies on an asynchronous event-drive architecture to help poser its goal of handling massive concurrent sessions. c. Apache Tomcat: Apache Tomcat is an open source JAVA servlet container that functions as a web server. Web servlets are the Java counterpart to other dynamic web content technologies such as PHP and ASP.NET. d. Node.js: Jode.js is a server-side JavaScript environment for network applications such as web servers. Node.js applies an event-driven architecture capable of asynchronous I/O. Node.js clearly part of the HTML, CSS, and JavaScript stack. e. Lighttpd: Lighttpd distinguishes itself with its low memory footprint, small CPU load, and speed optimizations. It uses an even-driven architecture, is optimized for a large number of paraller connections, and supports FastCGI, SCGI, Auth, Output-compression, URL-rewriting and many more features.
